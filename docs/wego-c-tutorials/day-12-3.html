Day 12 其他赋值运算符

C语言提供了一些特殊的赋值运算符，如 += -= *= /= %= >>= <<= &= ^= |=。

加法赋值运算符 = 

    #include <stdio.h>

    int main() {
        int n = 1;
        n += 1;
        printf("%d\n", n);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
2
