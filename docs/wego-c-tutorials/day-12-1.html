相等运算符 = = 

    float a = 1.5, b = 1.5;
    printf("%f eq %f is %d \n", a, b, a == b);

Tips == 是比较运算符，是二元运算符。
Tips 如果 a 和 b 相等，则 a == b 的值为真，即 1，否则为假，即 0。

不相等运算符 = 

    float a = 1.5, b = 1.5;
    printf("%f neq %f is %d \n", a, b, a != b);

Tips != 是比较运算符，是二元运算符。
Tips 如果 a 和 b 不相等，则 a != b 的值为真，即 1，否则为假，即 0。

逻辑且运算符 && 

    int a = 1, b = 1;

    int a_eq_b = a == b;
    int c_eq_d = c == d;
    printf("(%d eq %d) and (%f eq %f) is %d \n", a, b, c, d, a_eq_b && c_eq_d);

逻辑非运算符 

    #include <stdio.h>
        
    int main()
    {
        int a = 0, b = 1;
        printf("a is %d \n", a);
        printf("b is %d \n", b);
        printf("not a is %d \n", !a);
        printf("not b is %d \n", !b);
        printf("not ( a eq b ) is %d \n", !(a == b));
        printf("not ( a neq b ) is %d \n", !(a != b));
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
a is 0
b is 1
not a is 1
not b is 0
not ( a eq b ) is 1
not ( a neq b ) is 0

Tips >= 是比较运算符，是二元运算符。
Tips 一般情况下，a >= b 和 !( a < b) 的值是相同的。

请注意数学中的表达式和 C 语言中的表达式的区别。
例如：
数学中 1 <= a <= 10 虽然也是合法的 C 表达式，但是表达的含义不同。
当 a 等于 20 时，1 <= a 为 1， 所以 1 <= a <= 10 即 1 <= 10 也为真。
数学中的 1 <= a <= 10 用 C 语言表达应为： 1 <= a && a <= 10。

Summary 我们在本文介绍了C语言中常用的运算符。 其中包括 % ++ -- == != && || ! < > <= >=。
P.S. C语言还提供了一些其他的运算符， 包括表示数值正负的一元运算符 + 和 -， 获取数组元素的运算符 []， 取地址运算符 &， 解引用运算符 * (没错，和乘法运算符是同一个字符)， 访问指针指向的对象的成员的箭头运算符 ->， 用于改变默认优先级的括号运算符 ()， 用于位运算的运算符 ^ | & << >>， 以及扩充的赋值运算符 += -= *= /= %= >>= <<= &= ^= |=。
P.S. 事实上，逗号 , 也是C语言的运算符之一， 由于篇幅有限，我们会在以后介绍这些运算符。
