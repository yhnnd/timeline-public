Day 24 结构体指针

结构体指针是指向结构体的指针，声明和赋值的方法如下：
使用 
struct structType * pointer
声明一个结构体指针，指针名为 pointer，指向的结构体类型为 structType。
给该指针赋值，使用 & 运算符: 
pointer = &structName
structName 是一个 structType 类型的结构体
之前我们讲过，访问结构体的属性(成员)需要用点运算符 .。
例如 
structName.memberName
表示结构体 structName 的属性(成员) memberName。
那么我们如何访问结构体指针指向的结构体的属性呢？
现在我们介绍这个新的运算符 ->(箭头运算符)。
例如 
pointer->memberName
访问 pointer指向的结构体的 memberName 属性。
在访问结构体指针指向的结构体的属性的时候，还有一种书写比较复杂的方式 (*).，但是使用这种方式，我们更容易理解其原理：先解引用指针，得到其所指向的结构体，然后使用常规的点运算符 . 访问该结构体的属性。 
(*pointer).memberName
书写 (*pointer).memberName 时，() 是必不可少的。这是因为点运算符 . 的优先级高于解引用运算符 * 。 
    #include <stdio.h>
    #include <string.h>

    struct student {
        char firstname[32];
        char lastname[32];
    };

    int main() {
        // 声明 student 类型的结构体 stu 
        struct student stu;
        // 声明 student * 类型的结构体指针 pstu
        struct student * pstu;
        // 让指针 pstu 指向 stu
        pstu = &stu;
        // 给指针 pstu 指向的 student 结构体的 firstname 属性赋值
        strcpy(pstu->firstname, "Tom");
        // 给指针 pstu 指向的 student 结构体的 lastname 属性赋值
        strcpy(pstu->lastname, "Hanks");
        // 输出 stu 的 firstname 属性和 lastname 属性
        printf("%s %s\n", stu.firstname, stu.lastname);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
Tom Hanks
