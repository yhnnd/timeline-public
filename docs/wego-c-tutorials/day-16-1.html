在循环中嵌套判断语句

循环体中可以有循环语句和判断语句。组合使用循环和分支可以解决复杂的问题。

在密码学中有一种仿射加密算法(affine cipher)，该算法有两个密匙，设其中一个密匙为 a。 要求 a 是 1 到 26 中的某个整数，并且在 1 到 26 中能找到一个数 b， 使得 a 乘 b 除 26 的余数为 1。这时，我们称 b 为 a 在 26 下的逆。

设计一个程序，找出 1 到 26 中所有能选作密匙 a 的数字。

程序源代码如下：

    #include <stdio.h>

    int main() {
        int a, b;
        for (a = 1; a < 26; ++a) {
            for (b = 1; b < 26; ++b) {
                if (a * b % 26 == 1) {
                    printf("%2d 的逆为 %2d，\t%2d 可以被选作密匙。\n", a, b, a);
                }
            }
        }
        return 0;
    }

Tips 条件判断属于程序三大结构中的选择结构，又称分支结构。C语言中和分支结构有关的关键词有 if else 和 switch case。

自幂数 (pluperfect digital invariant) 或称阿姆斯特朗数 (Armstrong number) 是指一个 n 位数，它的每个位上的数字的 n 次幂之和等于它本身。 例如，当 n 为 3 时，有 1^3 + 5^3 + 3^3 = 153，153 即是 n 为 3 时的一个自幂数。 n 为 3 时的自幂数又称水仙花数 (Narcissistic number)。

设计程序，找出所有的水仙花数。

程序源代码如下：

    #include <stdio.h>
    #define cube(a) ((a) * (a) * (a))
        
    int main() {
        int n, a, b, c;
        for (n = 100; n < 1000; ++n) {
            a = n / 100;
            b = n % 100 / 10;
            c = n % 10;
            if (cube(a) + cube(b) + cube(c) == n) {
                printf("%d\n", n);
            }
        }
        return 0;
    }

Tips #define cube(a) ((a) * (a) * (a)) 是宏定义， 在其之后出现的 cube(...) 都会被替换为 ((...) * (...) * (...))。 代码中的 cube(a) + cube(b) + cube(c) 会被预处理器替换为 ((a) * (a) * (a)) + ((b) * (b) * (b)) + ((c) * (c) * (c))。 即对 a，b，c 分别求立方，再求和。

将数字的前后顺序颠倒后，数值不变的数，称为回文数(palindrome number)。设计程序，找出 1 到 1000 中的所有的回文数。

程序源代码如下：

    #include <stdio.h>
        
    int main() {
        int i, j = 0, k = 0;
        for (i = 1; i <= 1000; ++i) {
            if (i < 10) {
                printf("%4d", i);
                ++j;
            } else if (i < 100) {
                if (i / 10 == i % 10) {
                    printf("%4d", i);
                    ++j;
                }
            } else if (i / 100 == i % 10) {
                printf("%4d", i);
                ++j;
            }
            if(j % 10 == 0 && j != k) {
                printf("\n");
                k = j;
            }
        }
        printf("\n\nthere are %d palindrome numbers between %d and %d.\n", j, 1, 1000);
        return 0;
    }
