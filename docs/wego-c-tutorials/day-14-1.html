Day 14 do while 循环

do while 循环

do while 循环与 while 循环的不同在于：它先执行循环中的语句，然后再判断条件是否为真, 如果为真则继续循环；如果为假, 则终止循环。 因此，do while 循环至少要执行一次循环体。

语法如下：

    do {
        statement(s);
    } while (expression);

将 Day 14 中的代码用 do while 循环改写如下： 

    #include <stdio.h>

    int main() {
        int n = 1;
        do {
            printf("%2d 只青蛙 %2d 张嘴, %2d 只眼睛 %2d 条腿。\n", n, n, n * 2, n * 4);
            n++;
        } while (n <= 10);
        return 0;
    }
