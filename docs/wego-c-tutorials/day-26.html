Day 26 enum (枚举)

有时候，我们用不同整形数据代表不同的语义化标记。 因为数字所表示的意义不够明确，给审阅代码造成了很大困难。

C 语言提供了 enum (枚举) 的方式让我们定义多个语义化的值。 枚举类似于数组，但枚举中的任意两个值不能重复。 所以把枚举看作是集合更加恰当。

编译器对枚举的实现方式是： 枚举类型的数据以整形数据的存储方式存储。 每个枚举都有一个整形数据与之对应，它们的值相等。 或者可以这样理解：枚举类型的数据，就是取值范围有限的整形数据。

下面的例子可以很好地反映这一点： 

    #include <stdio.h>

    enum enumType {
        enumValue
    };

    int main() {
        enum enumType a = enumValue;
        int b = enumValue;
        printf("%s\n", a == b ? "true" : "false");
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
true

声明枚举集合时，程序开发者不需要显式指定每个枚举的值，编译器会采用默认方式给每个枚举设置取值。 一般情况下，默认的取值的方法是：从 0 开始采用递增方式依次给每个枚举设置取值。 所以，如果不通过显式指明枚举值的方式，枚举集合的所有值，是一个递增数列。

例如，在下面的代码中，从 Monday 到 Sunday 是 0 到 6 的递增数列。 

    #include <stdio.h>

    enum weekday {Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday};

    int main() {
        enum weekday weekday;
        for (weekday = Monday; weekday <= Sunday; ++weekday) {
            switch (weekday) {
                case Monday:
                    printf("%d Monday", weekday);
                    break;
                case Tuesday:
                    printf("%d Tuesday", weekday);
                    break;
                case Wednesday:
                    printf("%d Wednesday", weekday);
                    break;
                case Thursday:
                    printf("%d Thursday", weekday);
                    break;
                case Friday:
                    printf("%d Friday", weekday);
                    break;
                case Saturday:
                    printf("%d Saturday", weekday);
                    break;
                case Sunday:
                    printf("%d Sunday", weekday);
                    break;
                default:
                    printf("%d Error", weekday);
                    break;
            }
            if (weekday != Sunday) {
                printf("\n");
            } else {
                break;
            }
        }
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
0 Monday
1 Tuesday
2 Wednesday
3 Thursday
4 Friday
5 Saturday
6 Sunday

我们也可以显式指定枚举对应的整形数据的值。

程序设计者显式指定 Monday 的值为 1 之后， 编译器为 Monday 之后的枚举 (所对应的整形变量) 设置取值时所用的方式，仍然是递增的方式。 于是我们可以推断出：从 Monday 到 Sunday 是 1 到 7 的递增数列。 

    #include <stdio.h>

    enum weekday {Monday = 1, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday};

    int main() {
        enum weekday weekday;
        for (weekday = Monday; weekday <= Sunday; ++weekday) {
            switch (weekday) {
                case Monday:
                    printf("%d Monday", weekday);
                    break;
                case Tuesday:
                    printf("%d Tuesday", weekday);
                    break;
                case Wednesday:
                    printf("%d Wednesday", weekday);
                    break;
                case Thursday:
                    printf("%d Thursday", weekday);
                    break;
                case Friday:
                    printf("%d Friday", weekday);
                    break;
                case Saturday:
                    printf("%d Saturday", weekday);
                    break;
                case Sunday:
                    printf("%d Sunday", weekday);
                    break;
                default:
                    printf("%d Error", weekday);
                    break;
            }
            if (weekday != Sunday) {
                printf("\n");
            } else {
                break;
            }
        }
        return 0;
    }

2 Tuesday
3 Wednesday
4 Thursday
5 Friday
6 Saturday
7 Sunday
