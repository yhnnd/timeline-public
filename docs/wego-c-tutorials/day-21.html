Day 21 指针

什么是指针？

指针是一个变量，其值为另一个变量的内存地址。

就像其他变量或常量一样，您必须在使用指针存储其他变量地址之前，对其进行声明。

指针变量声明的一般形式为：

    
    type * ptr;

Tips 一个指针存储一个变量的地址，我们也可以称该指针指向一个变量。
Tips type 是指针指向的变量的类型，就像我们称某个变量为 type 类型变量一样，我们也可以称这个指针为 type 类型指针。
Tips ptr 是指针变量的名称。用来声明指针的星号 * 与乘法中使用的星号是相同的。

以下是有效的指针声明：

    
    char   * ptr_to_char;     /* 一个字符型的指针 */
    int    * ptr_to_int;      /* 一个整型的指针 */
    float  * ptr_to_float;    /* 一个单精度浮点型的指针 */
    double * ptr_to_double;   /* 一个双精度浮点型的指针 */

& 取地址符

以上的指针都没有被初始化，这意味着它们的值是不确定的。要想给一个指针赋值，就需要用的取地址符 &。

    
    int a = 1;
    int * ptr = &a;

NULL 指针

在变量声明的时候，如果没有确切的地址可以赋值，为指针变量赋一个 NULL 值是一个良好的编程习惯。赋为 NULL 值的指针被称为空指针。

NULL 是一个定义在标准库中的值为零的常量。

    #include <stdio.h>
    int main () {
        int  *ptr = NULL;
        printf("ptr 的值是 %p\n", ptr);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
ptr 的值是 0000000000000000

Tips NULL 必须是大写。在 stdio.h 中，NULL 是这样被定义的： 
#define NULL ((void *)0)
Tips 
%p
用于输出指针的值

* 解引用符

指针的值不是它指向的变量的值，而是它指向的变量的地址。要想通过指针获得它指向的变量的值，就需要用的解引用符 *。

* 运算符加在指针之前，可以返回该指针指向的变量的值。

设 ptr 是一个指向变量 a 的指针， 修改 ptr 的值只会修改它所指向的变量（也就是让 ptr 不再指向 a），不会修改 a 的值。 但是修改 * ptr 的值（等同于修改 ptr 所指向的变量的值），就是修改 a 的值。 

    #include <stdio.h>
    int main () {
        int n = 1024;
        /* 整型变量的声明 */
        int *ptr;
        /* 指针变量的声明 */
        ptr = &n;
        /* 设指针变量的值为 n 的地址 */
        printf("address of variable n: %p\n", &n);
        /* 变量的地址 */
        printf("address of variable n which pointer ptr points to: %p\n", ptr);
        /* 指针所指向的变量的地址 */
        printf("value of variable n which pointer ptr points to: %d\n", *ptr);
        /* 指针所指向的变量的值 */
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
address of variable n: 000000000061FE44
address of variable n which pointer ptr points to: 000000000061FE44
value of variable n which pointer ptr points to: 1024

--------------------------------
Process exited after 0.006051 seconds with return value 0
Press any key to continue . . .

空指针和 void * 类型指针

空指针（NULL 指针）是未指向任何变量的指针。 将暂时用不到的指针定义成空指针，可以防止一些程序错误（比如由野指针所导致的内存泄漏等错误）。

void * 类型的指针是有指向的指针，但它的指向的数据的类型暂时不确定。 有些情况下，使用该类型指针时需要强制转换。
