Day 24 结构体

C 语言中的结构体用于表达用户自定义的数据类型。结构体的关键字是 struct。

一个结构体可以有多个属性，属性的数据类型可以不同。访问结构体的某个属性时，要用点运算符（.）。 

    #include <stdio.h>
    #include <string.h>

    struct book {
        char title[64];
        char author[32];
        char press[32];
    };
    
    int main() {
        struct book Book;
        strcpy(Book.title, "The Kite Runner");
        strcpy(Book.press, "Bloomsbury Publishing PLC");
        strcpy(Book.author, "Khaled Hosseini");
        printf("%-20s %s\n", "book title:", Book.title);
        printf("%-20s %s\n", "publishing house:", Book.press);
        printf("%-20s %s\n", "author:", Book.author);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
book title:          The Kite Runner
publishing house:    Bloomsbury Publishing PLC
author:              Khaled Hosseini

运用 typedef 关键字，让用户可以自定义数据类型 book，这样在声明 book 类型的数据的时候，不需要写（也不能写） struct 关键字。 

    #include <stdio.h>
    #include <string.h>
    
    typedef struct {
        char title[64];
        char author[32];
        char press[32];
    } book;
    
    int main() {
        book Book;
        strcpy(Book.title, "The Kite Runner");
        strcpy(Book.press, "Bloomsbury Publishing PLC");
        strcpy(Book.author, "Khaled Hosseini");
        printf("%-20s %s\n", "book title:", Book.title);
        printf("%-20s %s\n", "publishing house:", Book.press);
        printf("%-20s %s\n", "author:", Book.author);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？

publishing house:    Bloomsbury Publishing PLC
author:              Khaled Hosseini
