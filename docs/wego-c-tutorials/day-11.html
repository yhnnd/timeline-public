Day 11 字符串

C 语言中字符串的表示方式是 char 型数组。

    #include <stdio.h>
    
    int main()
    {
        char name [] = "Hilary Bubbles";
        printf("Hello, %s.\n", name);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
Hello, Hilary Bubbles.

运用宏定义常量和 scanf 函数，将程序改写为可以输入数据的版本。

    #include <stdio.h>
    #define PRAISE "What a super marvelous name!"
    
    int main()
    {
        char name[40];
        printf("What's your name?\n");
        scanf("%s", name);
        printf("Hello, %s. %s\n", name, PRAISE);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
What's your name?
Tom Hanks
Hello, Tom. What a super marvelous name!
