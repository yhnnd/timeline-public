Day 5 C 语言注释(comment)

/* ... */ 用于给代码加文字注释，被注释的文本不会被编译。 

#include <stdio.h>

int main()
{
    /* 若要使用 printf 函数，必须引入 stdio.h 库文件 */
    printf("Hello, World! \n");
    return 0;
}
Tips stdio.h 中的 stdio 是 standard input and output 的缩写，h 是 header file 的缩写。 - yhn
Tips 给源代码加注释是一个良好的习惯，有助于代码的维护和修改。 - yhn

// ... 也可用于给代码加注释，被注释的文本不会被编译。 

#include <stdio.h>

int main()
{
    // 若要使用 printf 函数，必须引入 stdio.h 库文件
    printf("Hello, World! \n");
    return 0;
}
Tips // 是 C++ 风格的注释，也可用于 C 语言。 - yhn
Tips // 只能注释单行。 - yhn
