知识点

  • 所有的 C 语言程序都需要包含 main() 函数。 程序从 main() 函数开始执行。
  • /* ... */ 用于注释代码，其内容不会被编译。
  • printf() 用于格式化输出到屏幕。printf() 函数在 "stdio.h" 头文件中声明。
  • stdio.h 是一个头文件 (标准输入输出头文件) , #include 是一个预处理命令，用来引入头文件。
    当编译器遇到 printf() 函数时，如果没有找到 stdio.h 头文件，会发生编译错误。
  • return 0; 语句用于表示退出程序。
编译(compile) 就是把源代码翻译成目标程序

return 0; 和 system("pause"); 的作用都是防止程序闪退吗？
yhn
return 0; 不能防止程序闪退。
return 0; 是函数执行结束后返回给调用它的函数的值， 如果在 main 函数中写 return 0;，则是返回给系统 0。 一般情况下，返回 0 表示程序执行成功。 返回其它值（如 -1），则表示程序执行错误。
yhn
system("pause"); 加在 return 0; 之前可以防止程序闪退。 程序执行结束前会显示 Press any key to continue . . .。 这时，按任意键都可以结束程序。 使用 system 函数时需要引入头文件 stdlib.h 或 windows.h (如果开发环境是 Windows 操作系统)。 如下： 
    #include <stdio.h>
    #include <stdlib.h>

    int main()
    {
        printf("hello, %s\n", "world");
        system("pause");
        return 0;
    }

cgc186
那为什么 
void main
还能运行？
yhn
如果主函数为 void 类型或者不写 return 语句，某些 C 语言编译器会自动给主函数添加默认返回值 0。 在这种情况下，主函数返回值不规范，程序也能运行。
然而，如果使用的是最新版本的 GCC C 语言编译器，这种代码在编译时会报错： 
[Error] '::main' must return 'int'
不合法的主函数返回类型会导致编译失败。所以，建议主函数是 int 类型。
