yhn
这个代码是错的
因为 s 是指针，需要给指针分配内存
但是声明该指针时未初始化，也就没有为其分配内存
所以声明之后，不能用 = 将一个常量字符串赋值给它
而且 const 指针不能赋值给非 const 指针
可以改成 
const char * s = "abcd";
或者 
char s[] = "abcd";

yhn
你把所有在初始化时忘记设置长度的全都设置好长度 
char s[] = "123";
这种代码很危险
在声明 s 的时候, 没有指明长度, s 的长度默认是 4, 因为字符串最后有一个 '\0'
如果用 strcpy 等方法给 s 赋值更长的字符串, 就会发生内存溢出错误
我们在写代码时候, [] 里面不指明长度也可以, 但是最好这样写 
const char s[] = "123";
这样字符串初始化之后就不能被修改了
