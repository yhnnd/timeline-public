Day 6 C 语言变量(variables) 1 

    #include <stdio.h>

    int main()
    {
        int spider_num = 10;
        int spider_leg = 8;
        printf("%d spiders have %d legs.\n", spider_num, spider_num * spider_leg);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
10 spiders have 80 legs.

int 是变量类型，int 型变量代表整型变量，int 是 integer(整数) 的缩写。

= 不代表数学中的相等，而是赋值运算符，用于把该符号右边变量的值(rvalue)赋给左边的变量(lvalue)。

%d 是一种转换说明符(conversion specification)， 不会被直接输出，而是被参数列表中的参数一一替换。
第一个 %d 被 spider_num 替换了，第二个 %d 被 spider_num * spider_leg 替换了。
%d 中的 d 是 decimal 的简称，表示按十进制输出整型变量。 

 
    int spider_num = 10;
    int spider_leg = 8;

1. 上面两条语句可以简写成一条语句吗，怎么简写？ 

 
    printf("%d spiders have %d legs.\n", spider_num, spider_num * spider_leg);

2. 上面语句中的 \n 有什么作用？

3. 如果想输出 \ 和 %，代码该如何写？

Answer 1 两语句可以合并为 int spider_num = 10, spider_leg = 8;
Answer 2 \n 是换行的意思，n 代表 newline
Answer 3 printf("\\"); 可以输出一个 \ 字符， printf("%%"); 可以输出一个 % 字符。

int 类型的数据只能存储整数。如果要存储小数，请使用 float 或 double。

 
    const double E =        2.718281828;
    const double PI =       3.141592653;
    const double SQRT2 =    1.414213562;

给 int 类型变量赋值为小数，会发生什么情况呢？

 
    int a = 10.1;
    int b = 10.5;
    int c = 10.9;
    printf("a = %d\nb = %d\nc = %d\n", a, b, c);
以上代码编译的程序执行后会输出什么结果？ 
a = 10
b = 10
c = 10

--------------------------------
Process exited after 0.00611 seconds with return value 0
Press any key to continue . . .

不是四舍五入。那么，是向下取整吗？请看一下下面的代码。

 
    int a = -10.1;
    int b = -10.5;
    int c = -10.9;
    printf("a = %d\nb = %d\nc = %d\n", a, b, c);
以上代码编译的程序执行后会输出什么结果？ 
a = -10
b = -10
c = -10

--------------------------------
Process exited after 0.006296 seconds with return value 0
Press any key to continue . . .

也许结果和你所设想的不同。为什么不是 -11 呢？

因为 C 语言标准委员会规定，浮点数转化为整形数据时，向零取整。

也就是不考虑符号位（符号指的是正负号）向下取整。也可以说是正数向下取整，负数向上取整。

这样解释可能有些复杂，那么可以这样理解，对一个浮点数进行取整，就是取其小数点之前的数。
