Day 19 函数的声明，定义与调用

函数的声明

函数的类型即函数的返回值类型，函数的类型可以为空 (void)。函数的参数也可以为空。

语法如下：

    返回值 函数名 (参数类型 参数名 [,参数类型 参数名]);

例如：

    int max (int a, int b);

函数的定义

语法如下：

    返回值 函数名 (参数列表) {
        函数内容
    }

例如：

    int max (int a, int b) {
        int c = a;
        if (b > c) c = b;
        return c;
    }

上面的代码可用条件运算符改写成：

    int max (int a, int b) {
        return a > b ? a : b;
    }

函数的调用

函数的调用方法如下：

    #include <stdio.h>

    int max (int a, int b) {
        return a > b ? a : b;
    }

    int main() {
        int a = 4202, b = 4024, n;
        n = max(a, b);
        printf("%d\n", n);
        return 0;
    }

函数也可以直接作为其他函数的参数，例如：

    #include <stdio.h>
    
    int max (int a, int b) {
        return a > b ? a : b;
    }

    int main() {
        printf("%d\n", max(4202, 4024));
        return 0;
    }
