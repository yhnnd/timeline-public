Day 21 向函数传递指针

两种传参方式

2018年3月26日

在 C 语言中，向函数传递参数的方式有两种。

第一种方式是将参数的值传递给函数，这种传递方式就产生了两个概念，形参和实参。形参 formal parameters 是指函数定义时使用的参数，实参 actual parameters 是指函数被调用时实际用到的参数。 

 
    int func(int a) {
        a = 10;
    }

    int main() {
        int n = 1;
        func(n);
        printf("%d\n", n);
        return 0;
    }

上面的函数 func 被调用后， n 的值是多少？程序会输出什么？ 

 
    int func(int a, int b) {
        a = 10;
        b = 20;
    }

    int main() {
        int n = 1;
        func(n, n);
        printf("%d\n", n);
        return 0;
    }

上面的函数 func 被调用后， n 的值是多少？程序会输出什么？

经过实际运行，我们发现按值传递参数给函数，函数被调用后，参数的值不会改变（传递数组和结构体等复杂变量除外）。

如果我们想让函数改变参数的值，并且在函数外部也有效，我们该怎么做呢？

C 语言提供了两种传参方式，除了传值的方式之外，我们还可以传递地址给函数。 这种函数在声明和定义时，要指明参数是指针类型。

向函数传递指针参数

下面的函数 func 接受一个参数，该参数 ptr 为指针。函数 func 将 ptr 指向的整型变量的值加 10，因为在函数内外，指针的内容（也就是指针存储的变量的地址）是不变的，所以函数执行时所改变的，在函数外也有效。 

 
    #include <stdio.h>

    void func(int * ptr) {
        *ptr = *ptr + 10;
    }

    int main() {
        int a = 10;
        func(&a);
        printf("%d\n", a);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
20

下面的代码中，函数 func 的作用是交换两个变量的值。 

    #include <stdio.h>

    void swap(int * a, int * b) {
        int t = *a;
        *a = *b;
        *b = t;
    }

    int main() {
        int a = 10, b = 20;
        swap(&a, &b);
        printf("a = %d, b = %d\n", a, b);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
a = 20, b = 10

--------------------------------
