Day 12 基本运算符

赋值运算符 = 

    #include <stdio.h>
    
    int main()
    {
        int a;
        a = 1;  /* = 是赋值运算符，不是等于 */
        printf("a = %d \n", a);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？ 
a = 1

加法运算符 

    #include <stdio.h>
    
    int main()
    {
        int a, b;
        a = b = 1;
        /* 程序执行后会输出 1 plus 1 is 2 */
        printf("%d plus %d is %d \n", a, b, a + b);
        return 0;
    }
以上代码编译的程序执行后会输出什么结果？
