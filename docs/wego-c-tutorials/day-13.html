Day 13 表达式和语句

表达式

C语言中的表达式一种有值的语法结构，它由运算符将变量、常量、函数调用返回值结合而成。

实例

    /* 变量常量表达式 */
    a
    sum
    1
    0.5
    PI
                    
    /* 算术表达式 */
    a + b
    a - b + c
    a * b + 'A'
    a / b - c + 10
    i++ + i++ + ++i
                    
    /* 赋值表达式 */
    a = b
    a *= b += 20
    a = b = c = 10
    a = (b = 4) / (c = 2)
                    
    /* 逗号表达式 */
    1 + 2, 3 + 4
    (10, a * 5), a + 10
    
    /* 关系表达式 */
    x == y
    x <= y
    x != y
                    
    /* 逻辑表达式 */
    10 && 20
    0 || 1
    (a > b) && (!0)
                    
    /* 复合表达式 */
    x = ( y = (a + b), z = 10)

语句

语句指的是当程序运行时执行某个动作的语法结构。
表达式语句是最简单的一种语句，在表达式的末尾加分号就形成了一个表达式语句。 

x = 10;
y = 20;
x *= 10;    

1;
x + y; //这些语句也是合法的。语句被执行时，表达式求值，但不保存结果。
getchar(); //读取输入中的下一个字符，但不保存结果。

x++;
printf("hello, world\n");

在C语言中，一个语句的最后面要有分号 ;，如果没有分号，就不是语句。
表达式后面再加上分号，就成了表达式语句。

变量 变量名本身是一个表达式，表达式的值是变量当前的值。
常量 常量名本身是一个表达式，字面常量也是表达式。表达式的值是常量当前的值。
函数调用 对于返回值不为void的函数，对它的正确调用也是表达式。表达式的值为函数的返回值。
Tips 复合语句(compound-statement)也是语句的一种，其一般形式为：{block-item-list}。复合语句不是以分号结束，而是很多语句组成的。
