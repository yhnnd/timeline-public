Day 14 while 循环

while 循环

在指定条件为真时，while 循环会重复执行一个代码块。

语法如下：

    while (expression) {
        statement(s);
    }

expression expression 是循环执行的条件(condition)。
statement(s) statement(s) 是循环重复执行的代码块。
代码块 代码块是一对大括号 {} 内的代码片段。 

    #include <stdio.h>

    int main() {
        int n = 1;
        while (n <= 10) {
            printf("%2d 只青蛙 %2d 张嘴, %2d 只眼睛 %2d 条腿。\n", n, n, n * 2, n * 4);
            n++;
        }
        return 0;
    }

Tips expression 可以是任意的表达式。当条件为 true (任意非零值) 时执行循环。 当条件为 false 时，退出循环，继续执行紧接着循环的下一条语句。
Tips while 循环可能一次都不会执行。当条件为 false 时，会跳过循环体，直接执行紧接着循环的下一条语句。
Tips statement(s) 可以是几个语句组成的代码块，也可以是一个单独的语句。

Summary 我们在本文介绍了while 循环，之后我们会介绍 do while 循环和 for 循环。
