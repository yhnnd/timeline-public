假设从浏览器依次下载若干个文件，保存到本地硬盘的同一个目录（初始目录为空），需要对文件名进行处理，规则如下：

    generateFileName = (filenames) => {

        const names = [];

            if (count[name] == undefined) {

            } else {

                count[name] ++;

2 、 最大口罩产量

已知某工厂未来一段时间，每天口罩的生产产能依次记录于数组 productions，每天的生产计划依次记录于数组 plans，plans[i] 值为 1 表示生产，值为 0 表示休息。

由于疫情的影响，口罩需求增加，工厂决定修改生产计划进行赶工，在某 window 天内连续生产，即：在这 window 天中原计划休息的，开工生产。

在这window天之外的，生产计划保持不变

请制定合适的赶工方案，使得这段时间的总产量最大，并返回该最大值。

输入

第一个参数为整数序列 productions，800 <= productions[i] <= 1000
第二个参数为行为整数序列 plans，值仅为 0 或 1
第三个参数为 window

注：5 <= productions.length == plans.length <= 100000, 1 <= window <= productions.length

输出

一个整数，表示最大口罩生产量

样例1

复制输入：

[803, 804, 805, 802, 803, 804, 805, 804]
[1, 0, 1, 0, 1, 0, 1, 0]
3

4824

解释：

window 为3天。

可选的赶工方案（天数从1开始数）：
window 选择第1、2、3天：第2天从休息变为生产，这三天连续生产，其它天不变
window 选择第2、3、4天：第2、4天从休息变为生产
...
window 选择第6、7、8天：第6、8天从休息变为生产

选择第6、7、8天进行赶工，总产量最大，即第1、3、5、6、7、8天的产能之和，结果为 4824

class Solution {

    /**

     * 待实现函数，在此函数中填入答题代码

     * @param {number[]} productions

     * @param {number[]} plans

     * @param {number} window

     * @return {number}

     */

    getMaxQuantity = (productions, plans, window) => {

        const mountains = [];

        for (let i = 0; i < plans.length; ++i) {

            if (plans[i] === 0) {

                mountains.push(productions[i]);

            } else {

                mountains.push(0);

            }

        }

       

        const mm = [];

       

        mm[0] = 0;

        for (let i = 0; i < window; ++i) {

            mm[0] += mountains[i];

        }

       

        for (let i = 1; i < productions.length - window + 1; ++i) {

            mm[i] = mm[i - 1] - mountains[i - 1] + mountains[i + window - 1];

        }

       

        let windowBegin = 0, best = mm[0];

        for (let i = 1; i < mm.length; ++i) {

            if (mm[i] > best) {

                best = mm[i];

                windowBegin = i;

            }

        }

       

        let sum = 0;

        for (let i = 0; i < productions.length; ++i) {

            if (plans[i]) {

                sum += productions[i];

            }

        }

        return sum + best;

    };

}

3 、 Mini 订货系统

请实现一个Mini订货系统，提供订货、发货、查询功能：

·         OrderSystem() -- 系统初始化。

·         order(int customerId, string[] goods) -- 表示某客户的一次订货，goods每个元素表示一件商品，值为商品种类：

同一客户可以多次订货；单次订货，可对同一种类的商品订购多件。

·         deliver(string[] goods) -- 表示发货多件商品，goods含义同上：每个元素表示一件商品，值为商品种类

o    依次将每件商品按照订购的先后顺序发给订购这件商品的客户

注：用例保证发的商品一定被某客户订购过。

·         query() -- 查询并返回系统中未发货件数最大的客户Id；若存在并列，返回客户Id较小的；若所有客户都完成发货，返回 -1 。

class OrderSystem {

    constructor() {

        this.db = {

            // user 1: {

            //     good 1: 3,

            //     good 2: 1,

            // }

        };

        this.amountDb = {

            // user 1: 4,

        }

        this.orderDb = {

            // good 1: [user 1, user 2, user 1, user 1]

            // good 2: [user 1, user 2]

        };

    }

 

    /**

     * 待实现函数，在此函数中填入答题代码

     * @param {number} customerId

     * @param {string[]} goods

     */

    order(customerId, goods) {

        const user = "" + customerId;

        if (this.db[user] == undefined) {

            this.db[user] = {};

            this.amountDb[user] = 0;

        }

        for (const item of goods) {

            if (this.db[user][item] == undefined) {

                this.db[user][item] = 0;

            }

            this.db[user][item] ++;

            this.amountDb[user] ++;

            if (this.orderDb[item] == undefined) {

                this.orderDb[item] = [];

            }

            this.orderDb[item].push(user);

        }

    }

 

    /**

     * 待实现函数，在此函数中填入答题代码

     * @param {string[]} goods

     */

    deliver(goods) {

        for(const item of goods) {

            const user = this.orderDb[item].shift();

            this.db[user][item] --;

            this.amountDb[user] --;

        }

    }

 

    /**

     * 待实现函数，在此函数中填入答题代码

     * @return {number}

     */

    query() {

        const maxUser = {

            max: NaN,

            userId: NaN

        }

        for (const user in this.db) {

            const total = this.amountDb[user];

            const userId = parseInt(user);

            if (total > maxUser.max || (total > 0 && isNaN(maxUser.max))) {

                maxUser.max = total;

                maxUser.userId = userId;

            } else if (total == maxUser.max) {

                if (userId < maxUser.userId) {

                    maxUser.max = total;

                    maxUser.userId = userId;

                }

            }

        }

        // return JSON.stringify(this.db, ' ', 4);

        if (isNaN(maxUser.userId)) {

            return -1;

        } else {

            return maxUser.userId;

        }

    }

}
