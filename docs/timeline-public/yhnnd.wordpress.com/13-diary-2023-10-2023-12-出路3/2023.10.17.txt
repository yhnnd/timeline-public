//  'CM'            900
//  'M'             1000

void print (string symbol, int times) {
    for (int k = 0; k < times; ++k) {
        cout << symbol;
    }
}

int main() {
    const string symbols [13] = {"M", "CM", "D", "CD", "C", "XC", "L", "XL", "X", "IX", "V", "IV", "I"};
    const int values [13] = {1000, 900, 500, 400, 100, 90, 50, 40, 10, 9, 5, 4, 1};
    int n;
    cin >> n;// 3867
    int a = n;// 3867

    for (int j = 0; j < 13; ++j) {
        int times = a / values[j];
        a = a  - times * values[j];
        print(symbols[j], times);
    }
}

教會的牧者張崇峯牧師說: 杰斐逊总统剪掉后的《圣经》就84页，他是不是剪掉太多了，先知你怎么看这个问题？你打算留多少页？《保罗书信》能留下十行不？《彼得前后书》是肯定的全部都得剪掉了。

我說: 我不是先知。施洗者約翰是最後一位先知。《福音书》是语录体，必须得有问有答才行。魔鬼借着法利赛人提问，耶稣回答。如果把法利赛人的话全删了，那就没法读了。

教會的牧者張崇峯牧師說: 不用读，读什么啊？听你讲就行。

我說: 《彼得前后书》有什么问题吗？

教會的牧者張崇峯牧師說: 彼得写的啊，他是个大骗子你说的。完全不可信啊你说的。

我說: 我什么时候说过彼得是个大骗子？

教會的牧者張崇峯牧師說: 你说他没文化，写的东西不可信。那不就是大骗子。胡写一通。哗众取宠。

我說: 我说彼得是个渔夫，什么时候说过他是大骗子？

教會的牧者張崇峯牧師說: 你说他写的东西不可信，这不等于说他在说谎吗？承认吧，你比彼得伟大多了，没事，批评他几句也是看得起他。

我說: 我说《使徒行传》里写到他说的话不一定是他的原话，我说《使徒行传》是他写的了吗？《使徒行传》不是他写的。

教會的牧者張崇峯牧師說: 明白了，明白了。路加是大骗子。《使徒行传》是路加写的。那《路加福音》也不可信。

我說: 他写的书信也没有提到耶稣是《申命记》里说的那先知。

教會的牧者張崇峯牧師說: 那彼得原话是啥，耶稣告诉你没有，你给我们讲讲。

我說: 彼得的原话就在路加写的《使徒行传》上，彼得说那先知来的日子就是主的大日子，他没说那先知就是主。施洗约翰和主的日子是同一时期。所以不能根据彼得的那句话就断定彼得说的那先知是耶稣。

教會的牧者張崇峯牧師說: 对对对，日子日子。宋丹丹写了本《月子》，你写了本《日子》。所以我问你吗。你说啥就是啥。管他写的啥。让你看《圣经》真是屈才了，你能不能自己写一本？

我說: 要我说，主今天就来，今天世界就要被毁灭，您觉得可能吗？

教會的牧者張崇峯牧師說: 可能。你说就可能，你没说就没可能。关键在于你说没说。那你说了没有？到底来不来？

我說: 我不知道主的国是不是已经来了，所以才问您呢。我要是知道，我就不会问别人了。

教會的牧者張崇峯牧師說: 我不知道啊，我还没你知道的多呢，你问我有啥用。我说来了，是因为耶稣在梦里告诉我了。关键我也不信我自己啊，我不是信你吗？

我說: 跟从主是要背起十字架殉道的，不是天天扯闲篇开玩笑的。

教會的牧者張崇峯牧師說: 我的逻辑没问题啊。没开玩笑。我的逻辑哪里有问题吗？你捶死我，我不就殉道了吗？

我說: 我是基督徒，不是犹太人。你让我锤死你，我还能是基督徒吗？我又不是扫罗。

教會的牧者張崇峯牧師說: 你刚才捶我了，但是没锤死，所以你不是完整的基督徒了。但是這不重要，你是当代施洗约翰。你当然不是扫罗，你是当代施洗约翰。你是看守那日子的大先知。我领悟的对不对？到位不？

我說: 我不知道。我不是耶稣，我不知道祂的国是不是已经来了。

教會的牧者張崇峯牧師說: 你知道。你知道。你知道。

我說: 我若是施洗约翰，我定去不了天国。因为天国里最小的都比他大。

教會的牧者張崇峯牧師說: 没事，你去不了天国，也是地上最牛的。

我說: 地上最厉害的不是我们，是撒但。但我主已经胜过了世界。

教會的牧者張崇峯牧師說: 最小的都比他大，你说这句話是不是嫉妒施洗约翰，胡乱记载的？马太嫉妒施洗约翰，所以错误地记载了耶稣的话。我举一反三的能力咋样大师？

我說: 耶稣说的话，不可能是出自嫉妒，因为嫉妒来自那恶者。而且，我不是大师。

教會的牧者張崇峯牧師說: 马太嫉妒啊，是马太记载的这些话，马太是人不？所以他记下来的也不可信。马太还是个税吏，唯利是图的小人。他写下来的《马太福音》和上面说的耶稣的话，咋可能可信？

我說: 如果只有一部《福音书》记载了，也许不可信。但若是其他《福音书》都记载了，就可信。

教會的牧者張崇峯牧師說: 其他《福音书》是跟马太学的。以为马太记得对，没想到都抄错了。马可不但是人，还是个小人，受不了传道的苦，中间偷偷跑回家不干了，就是这种人写的《马可福音》，肯定是抄别人的。

我說: 如果您觉得错了，那就改过来。不要继续用错误的话语施洗。

教會的牧者張崇峯牧師說: 对对对，指着你改呢。我哪有你的本事？不敢造次。

我說: 我们是主的仆人，若我们得不到圣灵的启示，就没人能得到。我说的话不是出自嫉妒圣徒的心，而是出自爱主的心。若我们不比圣徒更爱主，就配不上主的恩典。

教會的牧者張崇峯牧師說: 说的真好。所以你说马太是不是嫉妒施洗約翰了？所以他才说「天上的都比施洗约翰大」？

我說: 若我们行主所行，在主里是无可指摘的，配同主和圣徒一起坐在父的右手边，世上的人称我是渔夫税吏又能如何？

教會的牧者張崇峯牧師說: 那彼得和马太说的又对了是吗？又可信了是吗？我跟不上你的节奏啊。

我說: 他们所说的，若是对的，那就是对的; 说的有错的，或者被人记载错了，我们心里也得清楚。

教會的牧者張崇峯牧師說: 完了，我这辈子也不知道哪部分对、哪部分错了，你也不告诉我。

我說: 古代是没有打印机的，抄错的地方多了去了。

教會的牧者張崇峯牧師說: 那我咋觉得现代打字错的更多呢？这一点大师我可不敢苟同。所以印刷的《圣经》不能看了。你也别看了。

我說: 我不是大师，但你说的有道理，所以打印的《圣经》也不能全信。打印的《圣经》和古代的《圣经》都要看。

教會的牧者張崇峯牧師說: 你是当代施洗约翰。

我說: 我们起初都相同，都是寻找神的人。但走着走着，有些人就越来越恪守律法和传统，忘记了神的诫命是叫我们彼此相爱，而不是让我们死在律法和传统的繁文缛节当中。当我们的弟兄们越来越像大祭司、利未人、拉比、法利赛人的时候，我们应该越来越像基督才行。

教會的牧者張崇峯牧師說: 打印的《圣经》不能看，电子的《圣经》更不能看，邪恶简直。一定要看手抄本。1000年以上的。
